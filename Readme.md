# Web Reader for VOICEVOX

> **Webサイトのテキスト読み上げを、あなたの好きな声で。**
> VOICEVOXエンジンの力を借りて、ブラウザ上のテキストを滑らかに読み上げるChrome拡張機能です。

---

## 主な機能

- **スマートなテキスト読み上げ**
  - ページ上のテキストを選択し、右クリックメニューまたはショートカット（`Alt+Shift+U`）で即座に再生。
  - マウスのサイドボタンで上記ショートカットを設定しておくと楽です。
- **VOICEVOXキャラクター対応**
  - オプション画面から、お気に入りのキャラクターを簡単に選択。速度調整も自由自在。
- **高度な再生キュー管理**
  - 複数のテキストを選択しても重ならず、順番に心地よく読み上げます。
- **クリーンな読み込み**
  - URLや不要な改行を自動で検出し、聞き取りやすいテキストに整形（クレンジング機能）。
- **プレミアムなUI設計**
  - **Shadow DOM**によるカプセル化を採用。どんなWebサイトのデザインも崩さず、安定した操作感を提供します。

## 動作要件

- **Google Chrome**（またはChromiumベースのブラウザ）
- **[VOICEVOX](https://voicevox.hiroshiba.jp/)** エンジン
  - ローカル環境で起動している必要があります（デフォルト: `http://127.0.0.1:50021`）。

## インストール方法

1. このリポジトリをダウンロードまたはクローンします。
2. Chromeで `chrome://extensions/` を開きます。
3. 右上の **「デベロッパーモード」** をONにします。
4. **「パッケージ化されていない拡張機能を読み込む」** を選択し、本プロジェクトのフォルダを指定します。

## 使い方

1. **VOICEVOXを起動** します。
2. Webページで読み上げたいテキストを **選択（ハイライト）** します。
3. **右クリックメニュー** から「Web Reader for VOICEVOXで読み上げ」を選択、または **`Alt + Shift + U`** を押します。
   - 再生中に再度実行すると、即座に停止します。
4. **キャラクターを変えたい時**
   - 拡張機能のアイコンまたはオプション画面から、利用可能なキャラクターリストを取得し、設定を保存してください。

## 技術仕様

- **Manifest V3 準拠**: セキュリティとパフォーマンスに優れた最新仕様。
- **Service Worker**: バックグラウンドでの安定したAPI通信。
- **Audio Memory Safety**: Data URLのメモリ解放を厳密に制御し、ブラウザの負荷を最小限に抑制。
- **Visual Isolation**: Shadow DOM (mode: closed) を使用し、サイト側のCSS干渉を完全に遮断。

# プライバシーポリシー

[人間学習]（以下、「提供者」といいます。）は、提供者が提供するChrome拡張機能「Web Reader for VOICEVOX」（以下、「本拡張機能」といいます。）における、ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます。）を定めます。

## 1. 収集する情報とその利用目的
本拡張機能は、Chrome Web Storeの「単一目的（Single Purpose）」ポリシーに従い、**Webページのテキスト読み上げ機能を提供することのみ**を目的として動作します。

提供者の管理する外部サーバーへ個人情報を送信・保存することはありません。すべてのデータ処理はユーザーの端末内（ローカル環境）で完結します。

### (1) テキストデータ
- **取得内容**: ユーザーがブラウザ上で選択し、読み上げを指示したテキストデータ。
- **利用目的**: 音声合成処理を行うため。
- **処理方法**: 当該テキストデータは、ユーザーの端末内で起動している「VOICEVOX」エンジン（`http://127.0.0.1:50021` 等）へ送信されます。提供者のサーバーや第三者のサーバーへ送信されることはありません。

### (2) 設定データ
- **取得内容**: 選択されたキャラクターID、読み上げ速度の設定値。
- **利用目的**: 次回利用時にユーザーの設定を保持するため（利便性の向上）。
- **保存場所**: ユーザーのブラウザ内のストレージ領域（`chrome.storage.local`）に保存されます。

## 2. データの最小化と秘匿性
本拡張機能は、読み上げ機能の実現に必要最小限の権限のみを要求し、それ以外のユーザーデータ（閲覧履歴、パスワード、クッキー等）にアクセスすることはありません。また、取得したデータを広告配信やプロファイリング、第三者への販売に利用することは一切ありません。

## 3. 第三者への提供
本拡張機能は、ユーザーの同意がある場合、または法令に基づく場合を除き、取得した情報を第三者に提供することはありません。
なお、音声合成のために連携する「VOICEVOX」ソフトウェアへのデータ受け渡しは、ユーザー自身のローカルネットワーク内で行われるものであり、インターネットを経由した外部への送信にはあたりません。

## 4. Google API データの限定的使用
本拡張機能の Google API（Chrome API）からの情報使用は、[Chrome Web Store 利用利用者データに関するポリシー](https://developers.google.com/chrome/web-store/program-policies?hl=ja#user_data)（限定的使用要件を含む）に従います。

## 5. 免責事項
本拡張機能からリンクや連携機能によってアクセスできる「VOICEVOX」ソフトウェア等の外部サービスにおける個人情報の取扱いについては、提供者は責任を負いません。

## 6. プライバシーポリシーの変更
提供者は、必要と判断した場合には、ユーザーに通知することなく本ポリシーを変更することができるものとする。変更後のプライバシーポリシーは、本ページに掲載したときから効力を生じるものとします。

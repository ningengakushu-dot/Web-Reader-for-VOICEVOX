# 利用規約

この利用規約（以下、「本規約」といいます。）は、[人間学習]（以下、「提供者」といいます。）が提供するChrome拡張機能「Web Reader for VOICEVOX」（以下、「本拡張機能」といいます。）の利用条件を定めるものです。利用者の皆様（以下、「ユーザー」といいます。）には、本規約に従って本拡張機能をご利用いただきます。

## 第1条（適用）
1. 本規約は、ユーザーと提供者との間の本拡張機能の利用に関わる一切の関係に適用されるものとします。
2. ユーザーは、本拡張機能をインストールまたは使用することにより、本規約に同意したものとみなされます。

## 第2条（本拡張機能の概要と要件）
1. 本拡張機能は、ユーザーがブラウザ上で選択したテキストを、ユーザーのローカル環境にインストールされた「VOICEVOX」ソフトウェアを通じて読み上げる機能を提供します。
2. 本拡張機能を利用するためには、別途「VOICEVOX」ソフトウェアのインストールおよび起動が必要です。VOICEVOXはVOICEVOX開発チームが提供するソフトウェアであり、本拡張機能とは独立しています。

## 第3条（VOICEVOX利用規約の遵守とクレジット表記）
1. 本拡張機能を利用して生成された音声の権利および利用条件については、VOICEVOXおよび各キャラクター（ボイスライブラリ）の利用規約に従うものとします。
2. ユーザーが本拡張機能を使用して生成した音声を動画配信、公開、または商用利用する場合、以下のクレジット表記を行う必要があります。
   - **表記例**: 「VOICEVOX:ずんだもん」「VOICEVOX:四国めたん」等

3. 各ボイスキャラクターには個別の利用規約が存在します。特定のキャラクターを継続的に使用・公開する場合は、必ず[VOICEVOX 利用規約ページ](https://voicevox.hiroshiba.jp/term/)を確認し、キャラクターごとの制限事項（禁止表現等）を遵守してください。
4. ユーザーは、生成した音声の利用に関わる一切の責任を自ら負うものとし、提供者は利用に伴う紛争やトラブルについて一切の責任を負いません。
